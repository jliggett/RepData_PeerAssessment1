Devise a strategy for filling in all of the missing values in the dataset

Use the mean for 5-minute interval

interval2steps <- function(interval) { stepsInt[stepsInt$interval == interval, ]$steps }

Fill in the missing data

activity <- data count = 0 for (i in 1:nrow(activity)) { if (is.na(activity[i, ]$steps)) { activity[i, ]$steps <- interval2steps(activity[i, ]$interval) count = count + 1 } } cat(“Total ”, count, “NA values were filled.\n\r”)
